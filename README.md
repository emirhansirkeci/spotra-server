## Description

API application to be developed for the backend of the <a href="https://github.com/emirhansirkeci/spotra" target="blank">Spotra</a> project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
